## vue-todo
Vue.js로 만든 Todo List 입니다.
- input의 value 공백 체크 후 List에 data 추가 및 삭제 기능 구현
- vuex를 이용한 상태관리로 store의 state가 업데이트 될때마다 localStorage에 data 변경
- https://craftedhaze92.github.io/vue-todo/index.html
